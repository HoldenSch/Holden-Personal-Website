Arcs iOS App GitHub

Tech Stack: Swift, SwiftUI, Firebase, Cloud Functions, JavaScript

Role: Solo Developer

Arcs is a native iOS application built with Swift and SwiftUI, featuring a robust Firebase backend infrastructure. This mobile app demonstrates modern iOS development practices with real-time data synchronization, cloud-based storage, and seamless user experience design. The application leverages Firebase's comprehensive suite of tools including Firestore for database management, Cloud Functions for server-side logic, and Firebase Authentication for secure user management.

The app showcases advanced iOS development techniques with a focus on performance, scalability, and user-centric design. Built entirely in Swift with SwiftUI for the user interface, the application provides a modern, native iOS experience that takes full advantage of the platform's capabilities. The Firebase integration enables real-time data updates, offline functionality, and cross-device synchronization, making it a comprehensive mobile solution.

With 97.3% Swift code and cloud functions written in JavaScript, this project represents a full-stack mobile development approach, combining native iOS development with cloud computing technologies. The app includes comprehensive testing suites with both unit tests and UI tests to ensure reliability and maintainability.
